# CoinTrade v2

Crypto Coin paper trading service. 

You can visit [Github Hosted Page](https://december-ok.github.io/CoinTrade/)!

API from [UpBit](https://docs.upbit.com/).

## development

```sh
git clone https://github.com/december-ok/CoinTrade

npm i

npm start
```

## Tech

- React
- Typescript
- WS
- Redux
- SCSS
- Chart.js
- Big.js
- CryptoJS
